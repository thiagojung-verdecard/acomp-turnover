# Notebooks

Exploratory and analysis notebooks live here. They are **not** production code.

## Rules

- Notebooks are for exploration only — extract stable code to `src/` before shipping
- Clear all outputs before committing (`jupyter nbconvert --clear-output`)
- Name notebooks: `NNN_<description>.ipynb` (e.g. `001_eda_raw_data.ipynb`)
- Add a markdown cell at the top with: purpose, dataset version, author, date
